### Linux, Storage & Kubernetes Interview Questions

### Experience

1.  Can you briefly explain your past experience and your current roles
    and responsibilities?


### Linux Troubleshooting

1.  Your Linux server is running very slowly. How would you approach
    troubleshooting this problem?
2.  While checking the output of the `top` command, what does the RSS
    column represent?


### Database

1.  Suppose you have a Cassandra cluster (or any database cluster), and
    for some reason the cluster is not functioning properly. How would
    you troubleshoot it?


### Storage & Disk Management

1.  Suppose you want to increase the size of an existing disk by 100 GB.
    How would you do it?
2.  How would you perform the same disk expansion using Terraform?
3.  After attaching a new disk to a virtual machine, how does the
    operating system recognize the new disk?
4.  What is the role of the SCSI bus in detecting newly attached disks?


### Kubernetes

1.  How would you perform the same disk expansion in a Kubernetes
    cluster?
2.  Suppose you need to add an additional 100 GB disk to each of the
    worker nodes in a Kubernetes cluster. How would you do that?
