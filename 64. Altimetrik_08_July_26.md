# Middleware, DevOps & Production Interview Questions

## Middleware

1.  How much experience do you have with middleware, and what middleware
    tools have you worked with?
2.  Can you give me an example of an Nginx reverse proxy configuration?
3.  How do you perform a zero-downtime reload in Nginx?

------------------------------------------------------------------------

## Automation & Scripting

1.  Which programming languages have you worked with for automation?
2.  Can you explain the difference between a Docker image and a Docker
    container?
3.  Can you write a Bash or Python script to troubleshoot a case where
    CPU utilization exceeds 95%?
4.  What does the `psutil` command/module do?
5.  Can you explain what line number 13 in your code does?
6.  Can you explain why you used the statement
    `echo "Current CPU Usage"` in your script?
7.  Can you write the same solution using Bash scripting?

------------------------------------------------------------------------

## Monitoring & Observability

1.  Can you explain the difference between SLI and SLO?

------------------------------------------------------------------------

## Production Support

1.  Can you tell me about the last production issue you resolved?
2.  In the production incident you explained, how did you verify that
    the issue was fixed?

------------------------------------------------------------------------

## Kafka

1.  How did you monitor Kafka consumer lag?
2.  How many Kafka brokers are there in your cluster?
3.  Have you configured any Kafka retention policies?

------------------------------------------------------------------------

## Jenkins

1.  How have you used Jenkins in your current project?
2.  Can you walk me through one Jenkins pipeline that you have
